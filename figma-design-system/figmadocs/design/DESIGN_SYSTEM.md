## üéØ **Best Practices**

### **DO:**

‚úÖ Use CSS custom properties (var(--primary))  
‚úÖ Use design tokens from globals.css  
‚úÖ Use semantic HTML (button, nav, main)  
‚úÖ Use custom icon components  
‚úÖ Test on mobile devices  
‚úÖ Check accessibility (WCAG 2.1 AA)  

### **DON'T:**

‚ùå Hardcode colors (#0d6efd)  
‚ùå Use text-* font-* classes (except when necessary)  
‚ùå Skip semantic HTML (<div onClick>)  
‚ùå Ignore mobile breakpoints  
‚ùå Forget keyboard navigation
